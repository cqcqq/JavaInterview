**Java是什么？**
 - Java是一门OOP（Object Oriented Programming）面向对象的编程语言，分为JavaSE、JavaEE、JavaME三个版本。
 - JavaSE开发桌面应用程序
 - JavaEE开发企业级应用程序
 - JavaSME开发嵌入式应用程序
 - 特点：
 - 简单、面向对象、分布式、多线程、解释型、跨平台：一次编写，到处运行。
 
**什么是对象？什么是面向对象？什么是类**
 - 对象是现实世界的实物，万物皆对象。
 - 面向对象是模拟现实世界实物。
 - 类是相似对象的统称。类由属性和方法组成。
 - 属性：类的基本信息。
 - 方法：类的行为。
 - 简单地讲，属性是名词，方法是动词。
